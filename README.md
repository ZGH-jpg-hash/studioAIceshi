# AI智能体开发教学项目

这是一个基于Python的AI智能体开发教学项目。

## 环境要求

- Python 3.6+
- pip

## 项目初始化

1. 创建虚拟环境：
```bash
python -m venv venv
```

2. 激活虚拟环境：
```bash
# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate
```

3. 安装依赖：
```bash
pip install -r requirements.txt
```

## 项目结构

```
studioAIceshi/
├── venv/              # 虚拟环境
├── requirements.txt   # 项目依赖
├── .gitignore        # Git忽略文件
└── README.md         # 项目说明
```

## 开发指南

- 使用Jupyter Notebook进行交互式开发
- 遵循PEP 8代码规范
- 使用black进行代码格式化
- 使用flake8进行代码检查
